02 // Graphic Design

津軽三味線

発表会で使用するのぼり、子引き出物のデザイン

Role

Visual Concept, Graphic Design

Duration

1 Week (2025.11)

Tools

Adobe Illustrator

Output

Stage Nobori Flag (のぼり旗)
Cookie Design (子引き出物)

01 // Overview

自身が12年間続けている「津軽三味線」の舞台演奏において、ステージの個性を際立たせ、演者と観客の空間的一体感を高めるための「のぼり旗」のビジュアル設計、および発表会に足を運んでくださった方々へ感謝を伝える「子引き出物」のデザインを担当しました。

02 // Creative Strategy

のぼり旗の設計にあたり、興行の伝統と圧倒的な存在感を放つ「大相撲ののぼり」の構成レイアウトをリサーチのベースとして参考にしました。演奏活動を共にしてきた同期ならではの「一人ひとりの個性」を言語化し、大胆なタイポグラフィとコントラストの強い配色へと昇華。これまでの伝統的な三味線のイメージを大切にしつつ、現代の舞台照明にも力強く映えるデジタル・ファブリックの親和性を意識したビジュアルを構築しました。

03 // Practical Implementation

Adobe Illustratorを使用し、実際に印刷・製作会社へ入稿可能な実寸大のベクターデータを作成。布地へ印刷された際の発色や照明による見え方の違いを計算し、文字の太さや境界線の処理をミリ単位で最適化しました。さらに、舞台の統一感を来場者に持ち帰ってもらうアプローチとして、子引き出物のためのオリジナルクッキーのデザインも企画・考案。クッキー自体のビジュアルが伝統芸能である舞台の世界観と連動するよう設計し、実践的なトータルプロデュースの感覚を養いました。
