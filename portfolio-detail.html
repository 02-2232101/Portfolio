03 // Web Design & Development

PORTFOLIO SITE

Self-Branding Web Portfolio

Role

UI/UX Design, Front-end Coding, AI Prompting

Duration

2 Week (2026.05)

Tech Stack

HTML / CSS / JavaScript / AI Assistant

Context

自己ブランディング・成果物集約用Webサイト

01 // Overview

大学での学びや自身の創作活動における成果物を統合し、スキルと姿勢をアピールするためのポートフォリオサイトです。

02 // AI Co-Development Process

本サイトのコーディングや、レスポンシブ対応におけるデバッグにおいて、生成AIを開発アシスタントとして活用する効率的なワークフローを試みました。

「大画面表示の際に見出し要素だけを左側にハミ出させて固定する」「スマホ環境のみロールバナーを停止させ、テキストを中央に静止させる」といった、自分のこだわりたいデザイン要件に対し、AIと対話を重ねながらコードの具体的なアプローチを模索。提案されたコードをそのまま鵜呑みにするのではなく、`clamp()` 関数や `calc()` などの数式、メディアクエリの優先順位といった裏側のロジックを自分自身で1行ずつ厳密に検証・理解した上で、自力で最適化を施して実装を完成させました。

03 // Engineering Value

AIの補助を的確に受けるために、自ら要件を論理的に言語化して指示を出すという、現代的な開発プロセスを経験できました。AIを単なる自動化ツールとしてではなく、自身の表現したいビジュアルを確実なコードへと落とし込むための「強力なパートナー」として主体的に乗りこなす、新しいものづくりの姿勢を形にした制作となりました。
